# 📚 ASP.NET Core API Training

This repository contains the projects I developed while following the **ASP.NET Core API** course by **Murat Yücedağ** on Udemy.

This was my **first complete course focused on API development**, where I learned the fundamentals of building RESTful APIs and consuming them from an ASP.NET Core Web UI application.

## 🚀 What I Learned

During this course, I gained hands-on experience with:

* Building RESTful APIs using ASP.NET Core
* Consuming APIs from an ASP.NET Core Web UI
* Applying **N-Layer Architecture (N-Katmanlı Mimari)**
* Using **AutoMapper** for object mapping
* Implementing **Authentication** and user security
* Managing users with **ASP.NET Core Identity**
* Creating and applying **Validation Rules**
* Using **View Components** to build reusable UI components
* Integrating **RapidAPI** services
* Sending emails through the application
* Developing CRUD operations between the API and the Web UI
* Managing HTTP requests and responses
* Working with Entity Framework Core for database operations

## 🛠 Technologies Used

* ASP.NET Core
* ASP.NET Core Web API
* ASP.NET Core MVC
* Entity Framework Core
* SQL Server
* AutoMapper
* ASP.NET Core Identity
* RapidAPI
* View Components
* Fluent Validation / Validation Rules
* Authentication

## 🎯 Course Outcome

This course helped me understand the fundamentals of API development and how a Web UI communicates with an API. It also strengthened my understanding of software architecture, authentication, validation, and building maintainable ASP.NET Core applications.

It represents an important milestone in my journey toward becoming a professional **.NET Backend Developer**.
